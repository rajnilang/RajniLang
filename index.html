To start with C version of Rajni-lang follow the steps below

1.To use the interactive repl go to the directory if cloned

Type "./rajni"

or

To execute a file written in Rajnilang use

"./rajni "

2.If you want to build Rajnilang from source

"cc -std=c99 -Wall rajni.c mpc.c -ledit -lm -o rajni"

Then follow step1.

Rajnilang is a dynamic language along the lines of javascript

Everything in Rajnilang is an object
