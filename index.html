Rajnilang is a dynamic language along the lines of javascript

Everything in Rajnilang is an object

Getting started

Welcome to the RajniLang wiki!

This is a step by step tutorial on getting started with RajniLang

Everything in rajnilang is an object and can be stored in a variable just like javascript. To set a variable use the following syntax

style {l} 23

This will create a variable named l with value 23. We can similarly set string or function as a value.

Multiple assignments can be done as below

style {l m} 23 24

which is equivalent to : "l=23 , m=24"

This will set l to the value 23 and m to the value 24

Arithmetic operations

lagalaga l 40

which is equivalent to

l=40

This will set the value 40 to l if doesn't exist it creates a new variable l and sets it.

kooti_kalichu_paru_sariya_varum_+ 2 3

which is equivalent to

2+3

kooti_kalichu_paru_sariya_varum_- 2 3

which is equivalent to

2-3

"sathigam" which is equivalent to multiplication

"prachodhagam" which is equivalent to division

"bayalagam" which is equivalent to percentage

Conditional statements

yen_vazhi_thani_vazhi(lagalagalagalaga 2 3) { andavan_solran_arunachalam_mudikran "true"} {    ena_pathi_uanku_theriyadhu "false" }

which is equivalent to if(2==3) { console.log("true") } else { console.error("false") }

Logical operations

anbuna_enanu_theriyuma 0 1

which is equivalent to 0 && 1

aarupadaiyappa 0 1

which is equivalent to 0 || 1

Comparision operations

adhigama_asai_padra_ambalayum 2 3

which is equivalent to 2 > 3

kamiya_kovapadadha_pombalayum 2 3

which is equivalent to 2 < 3

Basic print statement

andavan_solran_arunachalam_mudikran "Hello world"

console.log("Hello world")

ena_pathi_unaku_theriyadhu "Not found"

console.error("Not found")

Special feature

oru_thadava_sonna_noru_thadava_sona_madiri "Baasha"

which is equivalent to "Baasha" 100 times

Functions

Rajnilang has global and local scopes similar to javascript. It does not have the concept of classes and is a purely functional language.

style {l} (elam_maye {x y} {andavan_solran_arunachalam_mudikran (kooti_kalichu_paru_sariya_varum_+ 2 3)})

which is equivalent to

l = function(x,y) { console.log(x+y); }

To invoke this method use l 2 3 this will print 5

Examples which explain recursion

style {muthu} ( elam_maye {x y} { yen_vazhi_thani_vazhi (aarupadaiyappa (adhigama_asai_padra_ambalayum x y) (lagalagalagalaga x y) ) { andavan_solran_arunachalam_mudikran "true"} { muthu 1 1} })

muthu 0 1 to run this program

var muthu = function(x,y) { if((x>y)||(x==y)) { console.log("true") } else { muthu(1,1);
} }
